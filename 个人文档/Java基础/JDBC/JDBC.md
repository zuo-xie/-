### JDBC
- JDBC：Java DataBase Connectivity（Java语言连接数据库）
- JDBC本质：一套接口
- JDBC编程六步
    - 第一步：注册驱动
    - 第二步：获取连接
    - 第三步：获取数据库
    - 第四步：执行SQL语句
    - 第五步：处理查询结果
    - 第六步：释放资源
- 锁
    - 悲观锁
        - 行级锁：在`select`后面加`for update`
        - 事务必须排队执行，无法并发
    - 乐观锁
        - 支持并发
        - 不需要排队，需要一个版本号
        - 当版本号不相同时，事务回滚
